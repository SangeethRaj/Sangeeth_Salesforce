# ☁️Sangeeth-Salesforce-Journey

🚀 Apex Mastery Learning Path

Welcome to the Apex training repository. This guide is designed to help junior developers navigate the fundamental concepts of Salesforce development in a logical, step-by-step flow.

📂 Repository Structure
Each lesson is numbered sequentially using suffixes (`_L#`) within the class names to ensure the learning path is easy to follow.

🟢 Level 1: The Fundamentals (Data & Variables)

• L1: CoffeeShopMath - Basic data types (`Integer`, `Decimal`) and mathematical operations.

• L2: EmployeeProfile - Working with `Strings` and `Concatenation` to build data profiles.

🟡 Level 2: Control Flow (Logic & Decisions)

• L3: VipBouncer - Introduction to basic `if/else` statements.

• L4: GoldRoomLogic - Nested logic and comparison operators.

• L5: GuestListLogic - Using logical operators (`&&`, `||`) for multi-condition checks.

• L6: EliteGalaLogic - The Parentheses Rule. Learning operator precedence to group logic correctly.

• L7: MembershipKiosk - The `Switch` statement. Cleaner alternatives to long `if-else` chains.

🔴 Level 3: Collections (Bulk Data Handling)

• L8: GuestListController - Introduction to Lists. Adding and accessing items by index.

• L9: ListBouncerController - Advanced List methods: `.contains()`, `.indexOf()`, and manual `.remove()` logic.

• L10: GuestListSorter - Using `.sort()` to organize data alphabetically.

• L11: EmailManager - The For-Each Loop. How to process entire lists of data in a single block.

• L12: ListMethodGauntlet - The "Boss Fight." Combining all List methods to modify, search, and clean data.
