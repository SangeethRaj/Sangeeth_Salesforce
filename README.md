# ☁️Sangeeth-Salesforce-Journey

🚀 Apex Mastery Learning Path

Welcome to the Apex training repository. This guide is designed to help junior developers navigate the fundamental concepts of Salesforce development in a logical, step-by-step flow.

📂 Repository Naming Convention
Files: Named with a prefix (e.g., L1_CoffeeShop.cls) to keep the folder order clear.

Classes: Named descriptively (e.g., CoffeeShopMath) to maintain professional standards.

🟢 Level 1: The Fundamentals (Data & Variables)
L1: CoffeeShopMath – Basic data types (Integer, Decimal) and math.

L2: EmployeeProfile – Working with Strings and concatenation.

🟡 Level 2: Control Flow (Logic & Decisions)
L3: VipBouncer – Introduction to if/else statements.

L4: GoldRoomLogic – Nested logic and comparison operators.

L5: GuestListLogic – Logical operators (&&, ||).

L6: EliteGalaLogic – The Parentheses Rule (Operator precedence).

L7: MembershipKiosk – The Switch statement for cleaner code.

🔴 Level 3: Collections (Bulk Data Handling)
L8: GuestListController – Introduction to Lists (Ordered data).

L9: ListBouncerController – Advanced List methods (.contains, .remove).

L10: GuestListSorter – Using .sort() for alphabetical organization.

L11: EmailManager – The For-Each Loop (Processing data in blocks).

L12: ListMethodGauntlet – The "Boss Fight" combining all list techniques.

🔵 Level 4: Advanced Collections (Uniqueness & Relationships)
L13: UniqueGuestManager – Introduction to Sets (No duplicates allowed).

L14: DuplicateCleaner – The Constructor Trick (Instantly converting a List to a Set).

L15: MemberLookup – Introduction to Maps (Key-Value pairs).

L16: PhoneBookManager – Map methods (.put, .get, .containsKey).

🟣 Level 5 & 6: The Database (DML & SOQL)
L17: LeadCreator – Creating and Inserting your first Salesforce record.

L18: BulkLeadManager – The Golden Rule: Never put DML inside a loop.

L19: LeadFinder – Reading data using SOQL (The Salesforce Search Engine).

L20: LeadMassUpdater – The Mass Update Pattern (Fetch → Loop → Modify → Update).
